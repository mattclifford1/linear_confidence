# notebooks/

Development and exploration notebooks. Start here when working on the algorithm.

## Notebooks

| Notebook | Description |
|----------|-------------|
| `Gaussian.ipynb` | Core example on synthetic Gaussian data — best starting point |
| `Gaussian_sep.ipynb` | Separable Gaussian case |
| `Gaussian_logit_reg.ipynb` | Gaussian data with logistic regression classifier |
| `non_sep.ipynb` | Non-separable (overlapping) case |
| `moons.ipynb` | Two-moons synthetic dataset |
| `datasets.ipynb` | Run DELTAS on benchmark datasets |
| `SVM_supports.ipynb` | SVM support vector utilities |
| `SSL.ipynb` | Semi-supervised learning extension |
| `reprojection.ipynb` | Re-projection experiments |
| `downsample.ipynb` | Class imbalance via downsampling |
| `downsample_supports.ipynb` | Downsampling with support vector information |

## Suggested Order

1. `Gaussian.ipynb` — understand the basic algorithm on clean synthetic data
2. `Gaussian_sep.ipynb` — separable case behaviour
3. `non_sep.ipynb` — non-separable extension
4. `datasets.ipynb` — apply to real benchmark datasets
